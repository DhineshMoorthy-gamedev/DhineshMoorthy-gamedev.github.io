Genre: Utility / Tool

PlayFab Backend Integration

PlayFab Backend Integration is a Unity project that uses Microsoft PlayFab to manage game data and user profiles. I configured authentication methods, player profile structures, and cloud scripts while implementing secure user registration, login, and data storage through PlayFab APIs.

The project focused on optimizing network calls and data handling to reduce latency and improve performance. Leveraging PlayFab’s cloud features allowed me to build a scalable backend while concentrating on game design. This experience strengthened my skills in cloud-based game development, backend integration, and player analytics.

Genre: Utility / Tool
Engine: Unity
Team Size: Solo project
Platform: PC / VR (Oculus)

Fire Spread Shader

Developed a Fire Spread Shader in Unity, creating realistic flames that dynamically interact with the environment.

Designed using Shader Graph with texture mapping, particle systems, and color gradients, responsive to wind and environmental factors.

Implemented fire propagation to nearby flammable objects using colliders and triggers, enhancing realism and interactivity.

Optimized performance for VR, balancing visual quality with smooth operation through rigorous testing.

Genre: Visual Effects
Engine: Unity - Shader graph
Team Size: Solo Project
Platform: Compatible for all platforms
